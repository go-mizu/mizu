# 0162: Microsoft Teams Windows 11 Theme (team11)

## Overview

This specification defines the Microsoft Teams Windows 11 theme for the Mizu messaging blueprint. The theme replicates the modern Fluent 2 design language used in Microsoft Teams on Windows 11, featuring the signature Teams purple (#6264A7), Segoe UI typography, Mica-inspired backgrounds, and clean Fluent 2 design tokens.

## Layout Structure

### Main Layout
```
+--------------------------------------------------+
|                    Title Bar                       |
+----------+---------------------------------------+
|          |         Chat Header (48px)             |
|          +---------------------------------------+
| Sidebar  |                                        |
| (280px)  |         Messages Area                  |
|          |         (scrollable)                   |
|          |                                        |
|          +---------------------------------------+
|          |         Compose Area (56px+)           |
+----------+---------------------------------------+
```

### Sidebar Structure
- Width: 280px
- Header: User profile/search (48px)
- Chat List: Scrollable
- Each chat item: 48px height
- Avatar: 32px, rounded-full
- Hover: Background change with subtle transition

### Chat Header
- Height: 48px
- Avatar: 32px
- Presence indicator: 10px dot
- Action buttons: 32px square, subtle on hover

### Message Bubbles
- Max width: 65% of container
- Border radius: 4px
- Padding: 8px 12px
- Spacing between messages: 4px
- Group spacing: 16px

### Compose Area
- Min height: 56px
- Auto-expand up to 200px
- Border-radius: 4px
- Action buttons: 32px

## Component Specifications

### Chat List Item
```
Height: 48px
Padding: 8px 12px
Avatar: 32px circle
Content:
  - Name: Body 1 Strong, Foreground 1
  - Preview: Caption 1, Foreground 2 (truncate)
  - Time: Caption 2, Foreground 3
Unread badge: Brand Primary, 16px circle
```

### Message Bubble
```
Max Width: 65%
Padding: 8px 12px
Border Radius: 4px (8px for first/last in group)
Sent: Brand tint background, dark text
Received: Neutral background, dark text
Reactions: Attached below, 24px height
```

### Presence Indicators
| Status | Color |
|--------|-------|
| Available | #107C10 |
| Busy | #D13438 |
| Away | #FFB900 |
| Offline | #8A8886 |

### Compose Box
```
Min Height: 40px
Max Height: 200px
Border: 1px solid Border
Border Radius: 4px
Focus: 2px Brand Primary border
Actions: Attach, Emoji, GIF, Stickers (left)
Send: Brand Primary button (right)
```

## Animation

- Duration: 100ms (micro), 200ms (standard), 300ms (emphasized)
- Easing: cubic-bezier(0.33, 0, 0.67, 1) for enter
- Easing: cubic-bezier(0.33, 1, 0.67, 1) for exit

## Accessibility

- Minimum contrast ratio: 4.5:1 for text, 3:1 for large text
- Focus indicators: 2px solid with 2px offset
- Touch targets: Minimum 32px
- Screen reader support: ARIA labels on all interactive elements

## Theme Registration

1. Add `"team11"` to `Themes` array in `assets/embed.go`
2. Add `"team11"` to `THEMES` and `VIEW_THEMES` arrays in `assets/static/js/app.js`
3. Add theme preview card in settings page
