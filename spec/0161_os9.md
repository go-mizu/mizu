# Mac OS 9 iMessage Theme (iMOS9)

Theme ID: `imos9`

## Overview

This design specification describes a messaging application interface styled after Apple's Mac OS 9 Platinum appearance, combined with an iMessage-like layout. The theme recreates the characteristic visual elements of Mac OS 8/9 (1997-2001) including the gray window chrome, horizontal title bar stripes, 3D beveled buttons, and the Chicago/Charcoal system fonts.

## Design Philosophy

The Mac OS 9 Platinum appearance was the final iteration of the "classic" Macintosh interface before the transition to Mac OS X's Aqua. Key characteristics include:

- **Grayscale palette** with subtle color accents
- **3D beveled elements** creating depth through light/shadow
- **Horizontal stripe pattern** in title bars (the "racing stripes")
- **Small, square window controls** (close box, zoom box)
- **Bitmap-friendly typography** using Chicago and Charcoal fonts
- **High contrast** between interface chrome and content areas

### Window Controls

**Close Box** (left side):
- 11x11 px square with 1px black border
- Contains an X when hovered
- Red fill on hover (modern adaptation)

**Zoom/Collapse Boxes** (right side):
- 11x11 px squares with 1px black border
- Zoom contains resize indicator
- Collapse contains horizontal line

### Sidebar (Conversation List)

- Width: 200px (fixed)
- Border: 1px inset on right side
- List items: 40px height
- Avatar: 32x32px rounded square (2px radius)
- Alternating row colors optional

### Message Area

- White background (`#FFFFFF`)
- 12px padding
- Messages have 4px border-radius
- Sent messages aligned right with blue background
- Received messages aligned left with gray background

### Input Area

- Height: 44px
- Gray background matching window chrome
- Text field with inset border
- Send button as default button (thick border)
