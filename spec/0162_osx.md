# Mac OS X Aqua iMessage Theme (iMOSX)

Theme ID: `imosx`

## Overview

This design specification describes a messaging application interface styled after Apple's Mac OS X Aqua appearance (2001-2007), combined with an iChat-like layout. The theme recreates the characteristic visual elements of Mac OS X 10.0-10.4 including the iconic "lickable" gel buttons, translucent window chrome, pinstripe textures, and the signature red/yellow/green traffic light window controls.

## Design Philosophy

The Mac OS X Aqua interface, unveiled by Steve Jobs at Macworld 2000, revolutionized computer interface design. As Jobs famously stated, "it's liquid, one of the design goals was when you saw it you wanted to lick it." Key characteristics include:

- **Water-inspired aesthetics** with gel-like, translucent elements
- **High gloss buttons** with multiple gradient layers creating depth
- **Pinstripe texture** (alternating 1px lines) as background pattern
- **Traffic light controls** - red (close), yellow (minimize), green (zoom)
- **Rounded corners** on all elements (4-8px radius)
- **Extensive use of shadows** and reflections for depth
- **Blue as the primary accent color** with optional graphite mode
- **Lucida Grande** as the system font

## Responsive Considerations

While Mac OS X was desktop-only, we adapt for modern use:

- **< 600px**: Full-screen mode, hide sidebar, swipe navigation
- **600-900px**: Collapsible sidebar, compact toolbar
- **> 900px**: Full desktop layout with resizable sidebar
