# Mizu CLI

The `mizu` command-line interface is the official project toolkit for building and running Mizu applications. It provides a streamlined developer experience with project scaffolding, development server, and machine-readable output for CI/CD integration.

## Quick Start

```bash
# Create a new project
mizu new myapp --template api

# Navigate to your project
cd myapp

# Install dependencies
go mod tidy

# Run in development mode
mizu dev
```

## Commands

| Command | Description |
|---------|-------------|
| [`mizu new`](/cli/new) | Create a new project from a template |
| [`mizu dev`](/cli/dev) | Run the current project in development mode |
| [`mizu version`](/cli/version) | Print version information |

## Architecture

The Mizu CLI follows a simple command structure:

```
mizu <command> [flags]
```

Global flags can be placed before or after the command:

```bash
# Both are valid
mizu --verbose dev
mizu dev --verbose
```

## Design Philosophy

1. **Convention over Configuration** - Auto-detect project structure when possible
2. **Explicit is Better** - Provide clear flags for override when needed
3. **Machine-Friendly** - JSON output mode for all commands
4. **Fail Fast** - Clear error messages with actionable hints
