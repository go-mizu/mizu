Learning by examples is the best way to learn how to use **Mizu**.<br/>
*Mizu by Examples* is a practical, hands-on collection of runnable Go projects that guide you through building modern web applications step by step.

All examples are open source and available at [github.com/go-mizu/examples](https://github.com/go-mizu/examples).
You can clone the repository and start exploring right away:

```
git clone https://github.com/go-mizu/examples
cd examples
```

Each folder contains a small, focused Go program that demonstrates one clear idea: from serving pages to handling APIs, middleware, and real-time features.
Run any example instantly with a single command:

```
go run ./examples/001-hello-mizu
```

### What you will learn

By completing this series, you will learn how to:

- Build web applications using Go and Mizu
- Write clear and composable handlers and middleware
- Connect to databases and add caching for performance
- Secure your applications and handle errors properly
- Add real-time features and experiment with intelligent data-driven components

Begin with [Hello, Mizu](./001-hello-mizu) and learn step by step at your own pace.
