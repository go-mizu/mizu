### Install

```bash
go get github.com/go-mizu/mizu@latest
```

### Hello, Mizu

Create `main.go`:

```go
package main

import "github.com/go-mizu/mizu"

func main() {
    app := mizu.New()

    app.Get("/", func(c *mizu.Ctx) error {
        return c.Text(200, "Hello, Mizu!")
    })

    app.Listen(":3000")
}
```

### Run

```bash
go run main.go
```

Open [http://localhost:3000](http://localhost:3000) to see your app running.
