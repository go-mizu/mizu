The view engine provides built-in template functions that make your templates more expressive. This guide covers all available functions and how to add your own.

## Built-in Functions

The view engine includes these helper functions in addition to Go's standard template functions.

### Data Structure Functions

#### dict

Creates a map from key-value pairs:

```html
{{$user := dict "Name" "Alice" "Age" 30 "Admin" true}}
<p>{{$user.Name}} is {{$user.Age}} years old</p>
```

#### list

Creates a slice from values:

```html
{{$fruits := list "apple" "banana" "cherry"}}
<ul>
{{range $fruits}}
    <li>{{.}}</li>
{{end}}
</ul>
```

### String Functions

#### upper

Converts a string to uppercase:

```html
<span class="status">{{upper .Status}}</span>
<!-- "active" → "ACTIVE" -->
```

#### lower

Converts a string to lowercase:

```html
<div class="type-{{lower .Type}}">
<!-- "Featured" → "featured" -->
```

#### trim

Removes leading and trailing whitespace:

```html
<p>{{trim .Description}}</p>
```

#### contains

Checks if a string contains a substring:

```html
{{if contains .Email "@company.com"}}
    <span class="internal">Internal User</span>
{{end}}
```

#### replace

Replaces occurrences of a substring:

```html
{{replace .Content "\n" "<br>" -1}}
<!-- Replace all newlines with <br> -->
```

The third argument is the maximum number of replacements (-1 means all).

#### split

Splits a string into a slice:

```html
{{$parts := split .Tags ","}}
{{range $parts}}
    <span class="tag">{{trim .}}</span>
{{end}}
```

#### join

Joins a slice into a string:

```html
<meta name="keywords" content="{{join .Keywords ", "}}">
```

#### hasPrefix

Checks if a string starts with a prefix:

```html
{{if hasPrefix .URL "https"}}
    <span class="secure">Secure</span>
{{end}}
```

#### hasSuffix

Checks if a string ends with a suffix:

```html
{{if hasSuffix .Filename ".pdf"}}
    <span class="file-icon">PDF</span>
{{end}}
```

## Go's Standard Functions

Go's `html/template` package provides these built-in functions:

### Comparison Functions

| Function | Description |
|----------|-------------|
| `eq` | Equal |
| `ne` | Not equal |
| `lt` | Less than |
| `le` | Less than or equal |
| `gt` | Greater than |
| `ge` | Greater than or equal |

```html
{{if eq .Status "active"}}Active{{end}}
{{if gt .Count 10}}Many items{{end}}
{{if le .Stock 5}}Low stock{{end}}
```

### Logic Functions

| Function | Description |
|----------|-------------|
| `and` | Logical AND |
| `or` | Logical OR |
| `not` | Logical NOT |

```html
{{if and .LoggedIn .IsAdmin}}Admin controls{{end}}
{{if or .Error .Warning}}Alert{{end}}
{{if not .Disabled}}Enabled{{end}}
```

### Other Standard Functions

| Function | Description |
|----------|-------------|
| `len` | Length of array, slice, map, or string |
| `index` | Index into array, slice, or map |
| `print` | Format with default formatting |
| `printf` | Format with format string |
| `println` | Format with newline |
| `html` | HTML escape |
| `js` | JavaScript escape |
| `urlquery` | URL query escape |

```html
<p>{{len .Items}} items</p>
<p>First: {{index .Items 0}}</p>
<p>{{printf "%.2f" .Price}}</p>
```

## Adding Custom Functions

Add your own functions via the `Funcs` option:

```go
import (
    "html/template"
    "strings"
    "time"

    "github.com/go-mizu/mizu/view"
)

engine := view.New(view.Config{
    Dir: "./views",
    Funcs: template.FuncMap{
        // Simple function
        "greet": func(name string) string {
            return "Hello, " + name + "!"
        },

        // Function with multiple arguments
        "pluralize": func(count int, singular, plural string) string {
            if count == 1 {
                return singular
            }
            return plural
        },

        // Date formatting
        "formatDate": func(t time.Time, layout string) string {
            return t.Format(layout)
        },

        // Word count
        "wordCount": func(s string) int {
            return len(strings.Fields(s))
        },

        // Truncate with ellipsis
        "truncate": func(s string, length int) string {
            if len(s) <= length {
                return s
            }
            return s[:length] + "..."
        },
    },
})
```

### Using Custom Functions

```html
<h1>{{greet .Name}}</h1>

<p>{{.Count}} {{pluralize .Count "item" "items"}}</p>

<p>Published: {{formatDate .CreatedAt "January 2, 2006"}}</p>

<p>{{wordCount .Content}} words</p>

<p>{{truncate .Description 100}}</p>
```

## Common Custom Functions

### Date Formatting

```go
"formatDate": func(t time.Time, layout string) string {
    return t.Format(layout)
},

"timeAgo": func(t time.Time) string {
    duration := time.Since(t)
    switch {
    case duration < time.Minute:
        return "just now"
    case duration < time.Hour:
        return fmt.Sprintf("%d minutes ago", int(duration.Minutes()))
    case duration < 24*time.Hour:
        return fmt.Sprintf("%d hours ago", int(duration.Hours()))
    default:
        return fmt.Sprintf("%d days ago", int(duration.Hours()/24))
    }
},
```

```html
<time>{{formatDate .CreatedAt "Jan 2, 2006"}}</time>
<span>{{timeAgo .CreatedAt}}</span>
```

### Number Formatting

```go
"formatNumber": func(n int) string {
    // Simple thousands separator
    s := strconv.Itoa(n)
    if len(s) <= 3 {
        return s
    }
    // Insert commas
    var result []byte
    for i, c := range s {
        if i > 0 && (len(s)-i)%3 == 0 {
            result = append(result, ',')
        }
        result = append(result, byte(c))
    }
    return string(result)
},

"formatCurrency": func(amount float64) string {
    return fmt.Sprintf("$%.2f", amount)
},
```

```html
<p>{{formatNumber .Views}} views</p>
<p>Price: {{formatCurrency .Price}}</p>
```

### URL Helpers

```go
"urlEncode": url.QueryEscape,

"activeClass": func(currentPath, linkPath string) string {
    if currentPath == linkPath {
        return "active"
    }
    return ""
},
```

```html
<a href="/search?q={{urlEncode .Query}}">Search</a>
<a href="/" class="{{activeClass .CurrentPath "/"}}">Home</a>
```

### Safe HTML (Use Carefully!)

```go
"safeHTML": func(s string) template.HTML {
    return template.HTML(s)
},
```

```html
{{/* Only use with trusted content! */}}
{{safeHTML .TrustedContent}}
```

### Default Values

```go
"default": func(defaultVal, val interface{}) interface{} {
    if val == nil || val == "" || val == 0 {
        return defaultVal
    }
    return val
},
```

```html
<img src="{{default "/default-avatar.png" .Avatar}}">
```

## Best Practices

### 1. Keep Functions Pure

Functions should not have side effects:

```go
// Good: pure function
"double": func(n int) int { return n * 2 },

// Bad: has side effects
"logAndReturn": func(s string) string {
    log.Println(s)  // Side effect!
    return s
},
```

### 2. Handle Edge Cases

```go
"truncate": func(s string, length int) string {
    if s == "" || length <= 0 {
        return ""
    }
    if len(s) <= length {
        return s
    }
    return s[:length] + "..."
},
```

### 3. Use Pipelines for Readability

```html
<!-- Good: pipeline style -->
<p>{{.Bio | truncate 200 | upper}}</p>

<!-- Also fine: nested calls -->
<p>{{upper (truncate .Bio 200)}}</p>
```

### 4. Document Your Functions

```go
Funcs: template.FuncMap{
    // formatPhone formats a phone number as (XXX) XXX-XXXX
    // Example: "1234567890" → "(123) 456-7890"
    "formatPhone": formatPhone,

    // pluralize returns singular if count is 1, plural otherwise
    // Example: pluralize 1 "item" "items" → "item"
    "pluralize": pluralize,
}
```

### 5. Test Custom Functions

```go
func TestTruncate(t *testing.T) {
    tests := []struct {
        input    string
        length   int
        expected string
    }{
        {"hello", 10, "hello"},
        {"hello world", 5, "hello..."},
        {"", 5, ""},
    }

    for _, tt := range tests {
        result := truncate(tt.input, tt.length)
        if result != tt.expected {
            t.Errorf("truncate(%q, %d) = %q, want %q",
                tt.input, tt.length, result, tt.expected)
        }
    }
}
```

### 6. Be Careful with safeHTML

Only use `safeHTML` with content you completely trust:

```html
<!-- Safe: your own content -->
{{safeHTML .TrustedContent}}

<!-- Dangerous: user content - let Go escape it -->
{{.UserComment}}  <!-- Auto-escaped -->
```

## Complete Example

```go
package main

import (
    "fmt"
    "html/template"
    "strings"
    "time"

    "github.com/go-mizu/mizu"
    "github.com/go-mizu/mizu/view"
)

func main() {
    engine := view.New(view.Config{
        Dir:           "./views",
        DefaultLayout: "layout",
        Funcs: template.FuncMap{
            // String helpers
            "truncate": truncate,
            "slugify":  slugify,

            // Date helpers
            "formatDate": formatDate,
            "timeAgo":    timeAgo,

            // Number helpers
            "formatNumber":   formatNumber,
            "formatCurrency": formatCurrency,

            // Logic helpers
            "default":   defaultValue,
            "pluralize": pluralize,
        },
    })

    app := mizu.New()
    engine.Middleware(app)

    // ... routes

    app.Listen(":8080")
}

func truncate(s string, length int) string {
    if len(s) <= length {
        return s
    }
    return s[:length] + "..."
}

func slugify(s string) string {
    return strings.ToLower(strings.ReplaceAll(s, " ", "-"))
}

func formatDate(t time.Time, layout string) string {
    return t.Format(layout)
}

func timeAgo(t time.Time) string {
    d := time.Since(t)
    switch {
    case d < time.Minute:
        return "just now"
    case d < time.Hour:
        return fmt.Sprintf("%dm ago", int(d.Minutes()))
    case d < 24*time.Hour:
        return fmt.Sprintf("%dh ago", int(d.Hours()))
    default:
        return fmt.Sprintf("%dd ago", int(d.Hours()/24))
    }
}

func formatNumber(n int) string {
    return fmt.Sprintf("%d", n) // Simplified
}

func formatCurrency(amount float64) string {
    return fmt.Sprintf("$%.2f", amount)
}

func defaultValue(defaultVal, val interface{}) interface{} {
    if val == nil || val == "" {
        return defaultVal
    }
    return val
}

func pluralize(count int, singular, plural string) string {
    if count == 1 {
        return singular
    }
    return plural
}
```
