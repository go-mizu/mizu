# Welcome to Mizu

Mizu is a lightweight, composable web framework for Go that builds on the standard library's `net/http` package. Like water (水, "mizu" in Japanese), it flows naturally with Go's conventions while adding just enough structure to make web development enjoyable.

## Why Mizu?

<CardGroup cols={3}>
  <Card title="Built on net/http" icon="cube">
    No magic, no reflection, no code generation. Mizu uses Go 1.22+ ServeMux patterns and standard interfaces.
  </Card>
  <Card title="Composable Middleware" icon="layer-group">
    Over 70 production-ready middlewares for authentication, caching, rate limiting, and more.
  </Card>
  <Card title="Multi-Transport Contracts" icon="code">
    Define your API once, serve via REST, JSON-RPC, or MCP. Generate SDKs for Go, Python, and TypeScript.
  </Card>
</CardGroup>

## Quick Example

Here's all you need to create a working API:

```go
package main

import "github.com/go-mizu/mizu"

func main() {
    app := mizu.New()

    app.Get("/", func(c *mizu.Ctx) error {
        return c.JSON(200, map[string]string{
            "message": "Hello, Mizu!",
        })
    })

    app.Listen(":3000")
}
```

Run it:

```bash
go run main.go
```

Visit [http://localhost:3000](http://localhost:3000) to see your API in action.
