FAQ

Why Markdown instead of HTML?

HTML includes navigation, ads, scripts, and styling your agent doesn't need. Markdown strips that to headings, paragraphs, and links. About 80% fewer tokens.

Is it really free?

Yes. Open access, use the URL directly in your code or agent prompt.

How are JavaScript-heavy pages handled?

Pages that require JS fall back to a headless browser that runs the page fully before converting. Handles React, Vue, Angular, and any SPA.

Can I call this from the browser?

Yes. All endpoints include Access-Control-Allow-Origin: *. Call directly from browser JS.

What about login-protected pages?

Only publicly available URLs are accessible. Pages behind authentication or paywalls return an error.

What are the limits?

5 MB max response. 10s timeout (30s for browser rendering). Reasonable use is not rate-limited.
